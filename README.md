# Solo CRM
Pequeno CRM projetado e programado para empreendedores solo, com dashboard, domínio dos clientes, interações, anotações e gerenciamento de tarefas por meio de colunas (Kanban).

## Tecnologias
- Python
- Django
- PostgreSQL
- Chart.js
