# Solo CRM
SoloCRM é um sistema de CRM (Customer Relationship Management) desenvolvido especialmente para solopreneurs – profissionais que trabalham sozinhos e precisam de uma ferramenta simples e eficiente para gerenciar contatos, interações e acompanhar o funil de vendas.

## Funcionalidades
- **Gestão de Contatos (Clientes)**: Cadastro completo com nome, e-mail, telefone, empresa e estágio no funil (Novo, Contactado, Proposta, Ganho, Perdido).

- **Registro de Interações**: Adicione interações como ligações, e-mails ou reuniões vinculadas a um contato, com data e notas.

- **Anotações Detalhadas**: Cada interação pode conter várias notas, permitindo um histórico rico de cada conversa.

- **Painel de Controle (Dashboard)**: Visualize gráficos gerados com Chart.js para acompanhar rapidamente:
    - Distribuição de leads por estágio (pipeline)
    - Interações ao longo do tempo
    - Tipos de interação (e-mail, ligação, reunião)
    - Novos clientes adicionados recentemente

- **Interface Organizada**: As informações são apresentadas em uma grade responsiva, com cards individuais para cada gráfico e seções claras.

## Tecnologias Utilizadas

- **Backend:** Python, Django
- **Banco de Dados**: PostgreSQL
- **Frontend**: HTML, CSS, JavaScript, Chart.js
